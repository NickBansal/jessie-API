# Spotify music app

## Github commands

- `git pull` - Pulls all code from current branch
- `git add .` - Stages files ready to be committed
- `git commit -m "<YOUR GIT MESSAGE HERE>"` - Commits all changes ready to be pushed
- `git push`

## Getting started
`npm start`
